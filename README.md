# 📚 Study Resource Management System

A centralized platform designed to help students and educators organize, upload, and access academic materials efficiently. This system moves away from messy folder structures and brings study resources into a streamlined, searchable web interface.

---

## 🚀 Features

- **Resource Repository:** Upload and categorize notes and past papers.  
- **Real-time Search:** Find documents instantly using AJAX-powered filtering without refreshing the page.  
- **User Roles:** Separate dashboards for Admins (content management) and Students (viewing).  
- **Responsive Design:** Fully accessible on mobile, tablet, and desktop.  
- **Secure Authentication:** PHP-based login system with password hashing.
